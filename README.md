# پلتفرم سالن زیبایی

پکیج یک‌پارچه شامل **لندینگ + رزرو آنلاین + پنل مدیریت** برای سالن‌های زیبایی (ناخن، مو، پوست و …).

## تکنولوژی‌ها

| بخش | ابزار |
|-----|--------|
| بک‌اند | PHP 8.2+ |
| فرانت | React 18 + Vite |
| استایل | Tailwind CSS |
| دیتابیس | SQLite |

## نصب سریع

### پیش‌نیاز
- PHP 8.2+ با `pdo_sqlite`, `json`, `mbstring`
- Node.js 18+ (فقط برای بیلد فرانت)

### ۱. وابستگی‌ها

```bash
cd frontend
npm install
npm run build
```

> بک‌اند PHP نیاز به Composer ندارد (`bootstrap.php` اتولود را انجام می‌دهد).

### ۲. اجرای سرور توسعه

```bash
# از ریشه پروژه
php -S 127.0.0.1:8080 router.php
```

### ۳. نصب اولیه

**روش وب:** مرورگر → `http://127.0.0.1:8080/install/`

**روش خط فرمان (سریع):**
```bash
php scripts/install-cli.php "نام سالن" admin@example.com رمز123 "نام مدیر"
```

- نام سالن
- ایمیل و رمز مدیر (ورود پنل)

پس از نصب:
- **وبسایت:** `http://127.0.0.1:8080/`
- **پنل:** `http://127.0.0.1:8080/admin/`

## استقرار روی هاست

1. کل پروژه را آپلود کنید.
2. Document Root را روی `backend/public` بگذارید **یا** از `router.php` در ریشه با تنظیم مناسب استفاده کنید.
3. پوشه `backend/storage` باید قابل نوشتن باشد.
4. `/install/` را یک‌بار اجرا کنید.
5. پس از نصب، دسترسی به `/install/` را محدود کنید.

## قابلیت‌ها

- لندینگ با رنگ‌بندی و متن قابل تنظیم از پنل
- رزرو چند خدمت، انتخاب پرسنل، اسلات زمانی
- اعلان نوبت جدید برای پرسنل (مثلاً ناخن‌کار)
- پنل: نوبت‌ها، خدمات، پرسنل، مشتریان، تم
- نقش‌ها: مدیر (`super_admin`)، پرسنل (`staff`)، مشتری (`customer`)

## ساختار

```
Bu/
├── installer/       # ویزارد نصب
├── backend/         # API و SQLite
├── frontend/        # React (web + admin)
└── router.php       # سرور توسعه PHP
```

## توسعه فرانت

```bash
cd frontend
npm run dev:web      # پورت 5173
npm run dev:admin    # پورت 5174
```

پروکسی به API روی `8080` تنظیم شده است.

## پشتیبان‌گیری

کپی فایل: `backend/storage/database.sqlite`
